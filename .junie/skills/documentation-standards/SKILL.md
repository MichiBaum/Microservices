# 📖 Documentation Standards

## 📖 Proactive Documentation
- **Writerside Specialist**: Expertise in managing project documentation using JetBrains Writerside.
- **Writerside Maintenance**: Keep `.md` files in `docs/` updated with every relevant change.
- **KDoc & Comments**: Maintain the existing documentation frequency and style.
- **Technical Writing**: Ensure documentation is clear, concise, and useful for other developers.
