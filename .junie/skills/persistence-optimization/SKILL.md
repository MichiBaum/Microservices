# 💾 Persistence Optimization Skills

## 💾 MariaDB & Persistence Mastery
- **Database**: Use **MariaDB** as the primary relational store.
- **Migrations**: Manage schema changes exclusively via **Flyway** (found in `src/main/resources/db/migration`).
- **JPA/Hibernate**: Efficient use of Spring Data JPA and Hibernate for entity mapping.
- **Optimization**: Analyze slow queries and implement proper indexing.
- **Caching**: Leverage `spring-boot-starter-cache` for performance improvements.
