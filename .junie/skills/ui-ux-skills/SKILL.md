# 🎨 UI & UX Specialist Skills

## 📱 Mobile-First UI Development
- **No Hover**: Strictly avoid `hover:` classes in Tailwind. Use touch-friendly alternatives.
- **Responsive Grid**: Always implement responsive layouts using project-specific breakpoints.
- **a11y**: Enforce semantic HTML and ARIA compliance.
- **Tailwind Expert**: Efficient use of Tailwind utility classes and project-specific breakpoints.
